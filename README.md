# Infinity Client
This repository is one of the first Eaglercraft 1.12 hack clients. Press Right-Shift to open the menu.

How to use: 
1. Go to Releases section of this repository.
2. Click the HTML file you find there.
3. It should automatically download.
4. Go back to the main branch.
5. Find InfinityClientV1.js (It's literally the only file here).
6. Click into it and download it.
7. Go into your files
8. Open the HTML file you downloaded earlier.
9. Wait for it to load.
10. Once it loads, click "Upload Mod".
11. Select InfinityClientV1.js.
12. Click done.
13. Wait for game to boot up.
14. Join world or server.
15. Press Right-Shift to open menu.
16. Have fun!
